Lunar Lander

This is a realistic simulation of a lunar lander, where the player must control the descent of a spacecraft to land safely on the moon's surface. The game features realistic physics, including gravity and thrust, to create an immersive experience.

I worked with another software developer, Taden Marston, in pair programming to implement this project. We built it as part of our C++ course, focusing on encapsulation and design principles. We started with basic physics classes as the core of the simulation, and further simulations, such as the position, velocity, and acceleration classes. Once those classes were built and thoroughly tested, we tied them together with a callback function that draws a lunar lander and updates its position and angle based on the physics calculations. The project was a great learning experience in applying object-oriented design principles and encapsulation in C++.

Artillery Simulator

The Artillery Simulator is a physics-based simulation that accurately models projectile motion taking into account factors such as gravity, air resistance, and initial velocity. The user can adjust the angle of the howitzer by large amounts and by small amounts to fine-tune the projectile's trajectory. This will allow to use to hit targets at various distances.

This project, like the Lunar Lander Simulation, was developed as part of a C++ course, in pair programming with Taden Marston. This project also focuses on encapsulation and design principles. The simulation includes a howitzer and projectile classes, which encapsulate the physics calculations and rendering logic. The user interface allows for interactive control of the howitzer's angle and firing power, providing an engaging experience. In the development of this project, we reused the same physics classes from the Lunar Lander project, which allowed us to focus on the howitzer and projectile classes. This reuse of code not only saved development time but also ensured consistency in the physics calculations across both projects.

Technologies used: C++, Open Graphics Library, Accurate Physics Simulation

Chess

A fully functional Chess game implementation featuring all standard chess rules, including special moves like castling, en passant, and pawn promotion. The game includes move validation and a clean user interface.

This project was developed as part of a C++ course, focusing on object-oriented design principles. The chess game is built using a modular architecture, with separate classes for the board, pieces, and game logic. Each piece is represented by a class that encapsulates its movement rules and interactions with other pieces. The game logic handles the rules of chess, including move validation, en passant, and pawn promotion conditions. The user interface is designed to be intuitive, allowing players to easily interact with the game. The project also includes a comprehensive set of unit tests to ensure the correctness of the game logic and piece movements. This was another project developed in pair programming with Taden Marston, where we applied our knowledge of encapsulation and design principles to create a project with higher inheritance principles.

Technologies used: C++, Object-Oriented Design, Open Graphics Library

Orbital Simulator

An orbital mechanics simulator that demonstrates satellite motion around planetary bodies. The simulation takes into account gravitational forces and orbital mechanics principles to create realistic orbital paths.

This is another project developed in pair programming with Taden Marston, focusing on inheritance design principles. This simulator is the culmination of our work in the software design courses, where we applied our knowledge of inheritance and encapsulation to create a complex simulation. The project includes classes for orbital objects, satellites, pilotable ships, stars, etc. Each class encapsulates its own properties and behaviors, allowing for easy extension and modification. The simulation features a user interface that allows users to interact with the orbital objects, adjusting parameters such as velocity and position. The project also includes a comprehensive set of unit tests to ensure the correctness of the simulation logic and interactions between objects. This project was a great opportunity to apply our knowledge of inheritance and encapsulation in a practical context, demonstrating the power of object-oriented design in creating complex systems.

Technologies used: C++, Physics Simulation, Open Graphics Library

IPA Translator App

A translator application that converts text to International Phonetic Alphabet (IPA) notation. This tool helps users understand and learn proper pronunciation of words across different languages.

This project was a fun personal project that I developed over a weekend with the goal of borrowing an existing translation API and creating an iOS app in order to gain mobile development experience. It started with a simple interface that allowed users to input text and receive the IPA translation. I used the IPA Translation API to handle the translation logic, which provided accurate and reliable results. The app features a clean and intuitive user interface, allowing users to easily input text and view the IPA translation. There is an ability to translate from a handful of languages, including English, French, Spanish, and German. It is important to note that this app is not listed on the App Store, as the API was borrowed from an existing webstie and permission was not obtained for commercial use. However, it serves as a great example of my ability to develop mobile applications and work with third-party APIs.

Technologies used: Swift, IPA Translation API, GUI Development
